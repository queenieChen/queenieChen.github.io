Ctrl alt t
terminal
pwd显示当前所在目录

touch 1.txt新建一个文件

谁来执行linux命令？
ls -l

shell是一个程序，它接收来自键盘的命令并将它回显在桌面上，回车后，它就知道字符输入结束了，解析收到的命令，执行对应的程序。

shell会去path环境变量指示的位置寻找要执行的程序，然后启动该程序，传入参数。如果path路径下没有对应的程序，则需要使用相对路径，比如./hello。或者复制hello.exe到path指定的路径下去，然后再执行。
echo $path
等号前面的是变量，等号后面的是值
env

如何修改环境变量？
hello.exe放在路径/home/book

  1. 临时设置，只对当前终端有效，其他终端就无效了
    export PATH=$PATH:/home/Sbook
  2. 永久设置，修改~/.bashrc，这个只对当前用户SA有效
    （1）gedit ~/.bashrc
    （2）行尾添加export PATH=$PATH:/home/book
  3. 永久设置，修改/eStc/environment，这个对所有用户都有效
    （1）sudo gedit /etc/environment
    （2）在最后添加:/home/book

命令提示符：表示当前终端的状态

linux命令的格式：

命令 [选项] [参数]

ls

ls -l

ls -l /home

选项和参数是可选的，命令是必须的
选项是以 - 来指明的
命令，选项，参数是以空格隔开，多个空格视为一个空格

man ls可以查看ls的用法

1 directory 目录

3 list 列出

6 copy 复制
7 move 移动
8 clear 清除

pwd
print working directory

cd
change directory

cd ~表示家目录
cd .当前目录
cd ..切换到上一级目录
cd ../..切换到上上级路径
cd -切换到上一次路径

绝对路径

相对路径，一般以./和../构成

cd ../Videos

tab键自动补全

ls
list列出目录内容
!img

ls -l 显示目录下文件更详细的信息，文件权限，大小，最后修改时间等。l是long的缩写
ls -la显示隐藏文件，a是all的缩写
ls -lh将文件大小以k，kb，Mb，Gb来表示

mkdir
make directory

mkdir 目录名
mkdir -p 父目录/子目录

-p 连续创建多级目录，p是parent缩写

rmdir
remove directory

rmdir dir0不能删除非空目录
rm -rf 删除非空目录

touch用来新建文件
touch filename1

同一目录无法创建同名的文件，区分大小写

mv用来修改文件名，或者移动路径
mv oldfilename newfilename
mv 旧目录名 新目录名
mv 文件名 目录名

mv ~/flie1 ..将家目录下的file1文件移动到上一个路径

cp即copy
cp 源文件名 目标文件名
cp 源文件名 目标目录名

cp -r 目录名1 目录名2 表示将目录1下的所有文件都复制到目录2下去，-r表示递归

cp -i file2 dir1 如果dir1目录下有与file2同名的文件，-i表示同意覆盖

rm file2删除当前目录下file2文件

rm -i file2删除文件file2前，会先征求意见，y表示同意，n表示不同意

rm -r dir1表示递归删除目录dir1

rm -ir dir1

cat file2
cat -n file2显示文件内容，并在内容前显示行号
cat file2 file1串联打印

gedit 图形应用程序的编辑器

clear：刷新屏幕，会保留历史操作记录，只是向后翻一页

reset：重新初始化屏幕，清除历史
